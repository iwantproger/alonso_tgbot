# 🏎️ F1 2026 Telegram Bot

Бот с уведомлениями о гонках Формулы 1 2026 года.
Работает 24/7 на Railway (бесплатно).

---

## 📁 Структура проекта

```
f1_bot/
├── bot.py            ← Код бота
├── f1_2026.ics       ← Календарь F1 2026 (120 событий)
├── requirements.txt  ← Зависимости Python
├── Procfile          ← Команда запуска для Railway
├── railway.toml      ← Конфигурация Railway
└── .gitignore
```

---

## 🔑 Шаг 1 — Получи токен бота

1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя (например: `F1 2026 Notifications`)
4. Придумай username (например: `my_f1_2026_bot`) — должен заканчиваться на `bot`
5. BotFather пришлёт токен вида: `7123456789:AABBccDDeEFf...`
6. **Сохрани токен** — он понадобится на шаге 4

---

## 🐙 Шаг 2 — Выложи код на GitHub

1. Зайди на **github.com** → войди или зарегистрируйся
2. Нажми **"New repository"** (зелёная кнопка)
3. Название: `f1-bot` (или любое другое)
4. Выбери **Private** (чтобы токен не светился публично)
5. Нажми **"Create repository"**

### Загрузи файлы на GitHub:

**Вариант А — через браузер (проще):**

1. На странице репозитория нажми **"uploading an existing file"**
2. Перетащи все 5 файлов из архива: `bot.py`, `f1_2026.ics`, `requirements.txt`, `Procfile`, `railway.toml`
3. Внизу нажми **"Commit changes"**

**Вариант Б — через Git (если установлен):**

## 🚂 Шаг 3 — Деплой на Railway

1. Зайди на **railway.app**
2. Нажми **"Start a New Project"**
3. Выбери **"Deploy from GitHub repo"**
4. Авторизуй Railway в GitHub (кнопка **"Configure GitHub App"**)
5. Выбери репозиторий `f1-bot`
6. Railway автоматически создаст сервис

---

## ⚙️ Шаг 4 — Добавь токен в переменные окружения

1. В Railway кликни на созданный сервис
2. Перейди во вкладку **"Variables"**
3. Нажми **"New Variable"**
4. Введи:
   - **Key:** `BOT_TOKEN`
   - **Value:** `7123456789:AABBccDDeEFf...` ← твой токен от BotFather
5. Нажми **"Add"**
6. Railway автоматически перезапустит сервис

---

## ✅ Шаг 5 — Проверь что всё работает

1. Перейди во вкладку **"Deployments"**
2. Убедись что статус деплоя — **"Success"** (зелёный)
3. Кликни на деплой → **"View Logs"**
4. В логах должна быть строка: `🏎️  F1 2026 Bot запущен!`
5. Открой своего бота в Telegram и напиши `/start`

## 💬 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню с кнопками |
| `/subscribe` | Подписаться на уведомления |
| `/unsubscribe` | Отписаться |
| `/events` | Календарь: гонки / квалификации / все события |
| `/schedule` | Ближайшие предстоящие события |
| `/status` | Статистика подписчиков и уведомлений |

---

## 👥 Как добавить бота в группу или канал

1. Открой нужную группу → **Добавить участников**
2. Найди своего бота по username
3. Добавь его и дай права **"Отправка сообщений"**
4. Напиши в группе `/start`
5. Нажми кнопку **"Добавить этот чат"**

После этого уведомления будут приходить и в личку, и в группу.

---

## ⏰ Как работают уведомления

- **За 30 минут** до старта — предупреждение
- **Ровно в момент** старта — старт!
- Всё время отображается в **московском (МСК, UTC+3)**
- Всего 120 событий → до 240 уведомлений за сезон

## 🐛 Частые проблемы

**Деплой упал с ошибкой:**
- Проверь вкладку **Logs** → найди текст ошибки
- Убедись что добавил переменную `BOT_TOKEN`

**Бот не отвечает:**
- Убедись что сервис **запущен** (зелёный статус в Railway)
- Проверь что токен скопирован без лишних пробелов

**Бот не пишет в группу:**
- Убедись что добавил бота в группу
- Дай права на отправку сообщений
- Напиши `/start` в группе и подпишись через кнопку
